This is the source code for the Decisional States algorithm presented
in the article "Reconstruction of Epsilon-Machines in Predictive
Frameworks and Decisional States" by Nicolas Brodu.

You will find in this archive:

documentation.odt
  The documentation, start there.

include/
  This directory contains the algorithm implementation.
  The algorithm is fully templatised and can be used simply by
  including the appropriate files.

examples/
  Application examples described in the paper and the documentation
